# 玟珄個人網站
夏特稀開發的玟珄個人網站

## 如何編譯

### npm
透過以下指令，生成 ```dist/``` 資料夾，並且放入 gh_pages 的 branch 。

```bash
# 熱機制執行
npm i && npm run serve

# 編譯
npm run build
```

### Yarn-PKG

```bash
# 熱機制執行
yarn && yarn serve

# 編譯
yarn run build
```
